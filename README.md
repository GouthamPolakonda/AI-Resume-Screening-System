# AI Resume Screening System

A Flask-based AI Resume Screening System that:

- Uploads PDF and DOCX resumes
- Extracts resume text
- Performs NLP preprocessing
- Extracts candidate skills
- Matches resumes with job descriptions
- Ranks candidates using TF-IDF and Cosine Similarity

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- MySQL
- spaCy
- Scikit-learn
